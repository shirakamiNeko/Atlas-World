Atlas World 是一個把「AGI 當文明」來設計的長期計畫：  
我們不只討論模型能力，而是從憲法、價值漂移防禦、模擬倫理、身份協議等層面，設計一整套可實作的安全與治理內核。

This repository hosts the living documents and specs for Atlas World, including:
- Atlas World Constitution & amendments
- Atlas Safety Volume (identity, value-drift defence, simulation ethics)
- Multi-agent governance & protocol design
